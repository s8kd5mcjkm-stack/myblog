## Hugo 简介

Hugo 是一个用 Go 语言编写的静态网站生成器，以构建速度快而闻名。它支持：

- **Markdown** 编写内容
- **丰富的主题**生态
- **多语言**支持
- **代码高亮**
- **数学公式**渲染

## 代码示例

下面是一段简单的 Python 代码：

```python
def greet(name):
    return f"你好, {name}!"

print(greet("世界"))
```
